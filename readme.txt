魔津村-マツムラ-

1.ゲーム概要
敵を倒しながら旗までたどりついてください。

2.操作
方向キー：移動
Cキー:ジャンプ
Xキー:攻撃
Zキー:武器切り替え
Shift:ダッシュ

3.アイテム
・武器アイテム(武器の絵が描かれたプレート):
武器がもらえます
・回復(緑の珠):体力が回復します
・ジャンプUP(Jマーク):ジャンプ力が増えます
・ダッシュUP(Dマーク):ダッシュ力・回数が増えます
・コイン（コイン）:得点アイテムです

4.武器について
武器は3つまで持つことができ、zキーで切り替えることができます。
持っていない武器を取得すると数に空きがある場合は新たに武器が増え、
空きがない場合は今の武器と交換します。
Knife(短剣):直進する使いやすいオーソドックスな武器です。
Axe(斧):上方向に斧を投げます。対空攻撃のしやすい武器です。
Javelin（投げ槍）:速さのある槍を投げます。連射性能が悪めで攻撃力が高めです。
Bomb(爆弾):爆風で敵を吹き飛ばします。大きなノックバックが特徴です。
Boomerang（ブーメラン）:直進して少しの間とどまります。他の武器と合わせて使いましょう。

5.最後に
詰んだと思ったら壊せる壁を探しましょう。
